Sobre o OrganiZ

Nossa Missão

O OrganiZ é um sistema completo de organização pessoal desenvolvido para ajudar você a gerenciar seus compromissos, eventos e informações pessoais de forma eficiente e intuitiva.

Funcionalidades Principais

  • Sistema de Login Seguro: Autenticação robusta para proteger seus dados
  • Calendário Interativo: Visualize e gerencie seus eventos com facilidade
  • Gerenciamento de Eventos: Crie, edite e delete eventos com diferentes prioridades
  • Perfil Personalizável: Mantenha suas informações sempre atualizadas
  • Interface Responsiva: Funciona perfeitamente em qualquer dispositivo
  • Dados Persistentes: Suas informações são salvas com segurança no servidor

Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias modernas e práticas de desenvolvimento web:

  • Frontend: HTML5, CSS3 customizado, JavaScript ES6+
  • Backend: Node.js com JSON Server
  • Banco de Dados: JSON Database para desenvolvimento
  • Design: CSS Grid, Flexbox e animações customizadas
  • Responsividade: Mobile-first design

Como Começar

  1. Faça login com suas credenciais (ou use admin/123 para teste)
  2. Acesse o calendário para visualizar e criar eventos
  3. Edite seu perfil para personalizar suas informações
  4. Explore todas as funcionalidades disponíveis
